## URI

Uniform Resource Identifier 统一资源标志符
用来唯一标识互联网上的信息资源
包括URL和URN

### URL

http://user:pass@host.com:80/path?query=string#hash
---------------------------------------------------
1      2           3       4    5    6         7

1. schema 协议
2. 特定身份的认证 不安全，现在不用
3. 域名
4. 端口号
5. 路由
6. 搜索参数
7. hash   文档中的某一片段，锚点定位

以上此类格式都叫做URL，比如FTP协议
