Buffers是代码原始堆的分配额的数据类型，在JS中以类数组的方式来使用。
在全局可用，不需要require，可以将其视为是JS的另外一种类型(就像String Number一样)

// 分配255个字节
var buf = new Buffer(255);

// 第一个字节写入整形数据23
buf[0] = 23;
