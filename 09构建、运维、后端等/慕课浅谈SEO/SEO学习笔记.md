# 浅谈搜索引擎——SEO

## SEO 与 SEM 的区别

* SEO Search Engine Optimization 搜索引擎优化
* SEM Search Engine Marketing 搜索引擎营销

* IP 独立IP访问的用户
* PV 页面浏览量或点击量
* UV 独立访客数 

## SEO排名机制

* 搜索引擎蜘蛛

1. 百度搜索网站登录口

### 优化三要素

1. 标题 - title
2. 关键字 - keyword
3. 描述 - description

外链（友情链接）是提升权重的方式

注意事项

1. 和内容相近的网站交换
2. 连接交换形式多样
3. 灌水式外链

**内容质量**

1. 更新
2. 质量
3. 原创
4. 关键词密度（2% ~ 8%）
