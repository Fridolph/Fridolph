### 无处不在的跳转

按 ESC 会返回到代码窗口

小工具之间的跳转：

alt + 1 项目管理窗口
alt + 2 代码标记窗口
alt + 4 代码运行窗口

---

项目之间的跳转：
ctrl + alt + [ 上一个项目 任务栏窗口
ctrl + alt + ] 下一个项目 任务栏窗口

文件之间的跳转：

Navigate -> last edit location
