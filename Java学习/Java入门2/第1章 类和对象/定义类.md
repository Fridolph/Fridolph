## 成员变量和局部变量

1. 成员变量

在类中定义，用来描述对象将要有什么

2. 局部变量

在类的方法中定义，在方法中临时保存数据

## 成员变量和局部变量的区别

* 作用域不同

布局变量的作用域仅限于定义它的方法

成员变量的作用域在整个类内部都是可见的

* 初始值不同

Java会给成员变量一个初始值

Java不会给局部变量赋予初始值

* 就近原则

两类变量同名时，局部变量具有更高的优先级
